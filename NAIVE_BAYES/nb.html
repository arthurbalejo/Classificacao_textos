Naive Bayes NB

um classificador probabilstico que utiliza o Teorema de Bayes para prever a probabilidade de uma instncia pertencer a uma determinada classe. Ele chamado de naive (ingnuo) porque assume que todas as caractersticas (ou atributos) so independentes entre si, o que raramente verdade na prtica, mas simplifica muito os clculos.

from sklearn.feature_extraction.text import CountVectorizer
import pandas as pd
from sklearn.model_selection import cross_val_score, cross_val_predict, StratifiedKFold

from sklearn.naive_bayes import MultinomialNB
